Deletes redundant data from the WordPress database, speeds up your site, and reduces the database size!

== Description ==
This plugin will delete the following redundant data from the WordPress database:

* Comments That Failed Moderation.
* Orphan Commentmeta.
* Orphan Postmeta.
* Orphan Relationships.
* Post Auto-Drafts.
* Post Drafts.
* Post Old Slugs.
* Post Revisions.
* Spam Comments.
* Transient Options.
* Trash Comments.

Optionally, it will also optimise the database, saving space and increasing performance.
					
== Installation ==
The easiest way to enjoy LDW Clean is to login to your WordPress dashboard, go to Plugins > Add New,
search for LDW Clean and click 'Install Now'.

You can also download the zip file from this page and upload it from the Plugins > Add New > Upload page.

== Frequently Asked Questions ==
= Usage =
1. Dashboard > Settings > LDW Clean.
1. Select the areas you want to optimise.
1. Click 'Clean Database'.
